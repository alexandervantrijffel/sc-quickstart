QuickStart object creation wizard - Start a program


Path

In the Start a program-dialogscreen the path to a executable file has to be specified. The path can be entered manually or selected from the Select program-dialog. Click on the browse icon to display the Select program-dialog which shows all registered applications on your computer. Click on the other-button to browse to applications not listed in the Select program-dialog.

 

Note:
Relative paths are not allowed.

 

Arguments

Arguments are optional command line options to be sent to the launched program.
Examples are -o or -v.
