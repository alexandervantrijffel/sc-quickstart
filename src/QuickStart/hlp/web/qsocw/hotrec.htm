QuickStart object creation wizard - Record a hotkey


A hotkey is a key combination like CTRL+5 or SHIFT+ALT+F7 which can be linked to a QuickStart object. Record a hotkey to start a QuickStart object with only one single keystroke.

 

Record

Click on the record-button to display the 'Record hotkey'-dialog. Make sure that the 'Record hotkey'-dialog has the focus and enter the desired key-combination.

Do not enter a common Windows hotkey to make sure that your hotkey will not be used by another application. Suppose that you've created a QuickStart-object for launching a calculator using hotkey CTRL+P. Many applications support CTRL+P for printing a document. SC-QuickStart will intercept all CTRL+P keystrokes: printing using CTRL+P won't be possible anymore.

See Common Windows Hotkeys for a list of frequently used key-combinations.

Examples of rarely used key-combinations are:

  • CTRL+1
  • CTRL+ALT+F2
  • ALT+F2

Note:
SC-QuickStart uses CTRL+F1 for hiding and showing SC-QuickStart's mainscreen.
