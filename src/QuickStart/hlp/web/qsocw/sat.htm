QuickStart object creation wizard - Start a task


Task

Select the task that you want to quickstart. SC-QuickStart supports the following tasks:

Task Description Options
Dial default internet connection Create a default dialup connection. -
Hangup internet connection

Disconnect the currently active dialup connection.

 -
Shutdown or logoff

Shutdown or logoff or restart Windows. No confirmation dialog will be displayed.

 Shutdown - Logoff - Reboot
Eject CD Eject a closed CD-ROM-tray. All CD-Rom drives - Drive letter
Close CD Close an open CD-ROM-tray. All CD-Rom drives - Drive letter
Show Control Panel Open or explore the Windows Control Panel. Open - Explore
Start screensaver Activate the default screensaver. -
Increase Display Resolution* Increase the current display resolution with one step. Example: if the current display resolution is 800x600 the resolution will be changed to 1024x768. -
Decrease Display Resolution* Decrease the current display resolution with one step. Example: if the current display resolution is 1024x768 the resolution will be changed to 800x600. -
Close all programs When executed, this task sends a close command to all active desktop programs. Note: some buggy applications might handle the close command incorrectly and will crash.  
Empty Recycle Bin** Empty the Windows Recycle bin. If the recycle bin holds any items a confirmation dialog will be displayed. -

* available on Windows 2000 and XP
** This command is rotating. If screen output is lost, keep increasing or decreasing the display mode until you've reached a valid screen resolution.

 

Test

Click on the test button to check whether the task satisfies your needs.
