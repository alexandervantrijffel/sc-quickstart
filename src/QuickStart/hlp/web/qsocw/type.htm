QuickStart object creation wizard - Type



The first screen of the wizard let's you choose the kind of QuickStart object that you want to create:

 

Type Description
Start a program Launch an executable file (.exe) from disk.
Start a task

Start a common Windows task.

Available tasks are:

  • empty recycle bin
  • dial/disconnect internetconnection
  • open control panel
  • shutdown restart windows
  • Eject/close cd-rom tray(s)
Open a folder

Explore or open a directory with theWindows Explorer.
Open a document Open a document file with the default program handler or open a website with the default browser.

 

Click on the desired type to continue.
