QuickStart object creation wizard - Open a document


Path

Enter either a full path to a document or the URL of your favourite website or ftp-server.
Examples are:

  • http://www.soft-central.net
  • ftp://ftp.somedomain.com
  • c:\Program Files\SC-QuickStart\SC-QuickStart.chm

Enter the path manually or use the browse-button to select a document within the browse-dialog.

 

Note:
Relative paths to documents are not allowed.
