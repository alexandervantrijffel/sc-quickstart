Features

 

Tasks that can be started with ease with SC-QuickStart:

  • Execute applications
  • Open documents
  • Print documents
  • Open webpages
  • Print webpages
  • Open or explore folders (directories)
  • Empty recycle bin
  • Open/close CD-Rom drives
  • Shutdown/reboot computer

QuickStart tasks using:

  • Key combinations
  • Systemtray menu
  • Mainscreen of SC-QuickStart
