Order information


SC-QuickStart is protected from piracy by a software licensing system. SC-QuickStart can be used for a limited free trial period, which will give you sufficient time to fully evaluate the value of the product. The trial period is shown on the start-up splash screen. We do not limit the function of the software during this trial period, however, once the period has expired, you will not be able to use the trial version or another installation of the trial software on the same computer.

A single-user license is $15 US Dollars. Contact sales@soft-central.net for multi-volume discount options.

Registering SC-QuickStart

Once you have completed the sales transaction you will be emailed a registration number. start SC-QuickStart and press the button labeled 'register' on the welcome screen. If you miss this opportunity, close the application and repeat the process. You will be requested to enter your name and registration number we sent you in the order confirmation email. Enter your name and registration number EXACTLY as they appear in the e-mail that you receive from Soft-Central.
