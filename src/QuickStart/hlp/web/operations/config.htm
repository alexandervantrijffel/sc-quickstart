Configure SC-QuickStart


Click on the icon with the gearwheels to display the configuration dialog.

 

Start automatically when windows starts

Clear the check mark if you want to start SC-QuickStart manually.

 

Show main screen on startup

Clear the check mark if you do not want the main screen to be displayed at program startup.

 

Remove all QuickStart objects

Removes all configured QuickStart objects. WARNING: This operation cannot be undone.

 

Load default QuickStart objects

Loads the QuickStart objects that are available when SC-QuickStart is executed the first time. The default QuickStart objects can either be added to your current configuration or you can choose to overwrite the current configuration.
