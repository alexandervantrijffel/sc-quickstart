Modifying QuickStart objects


Right-click a QuickStart object in the main screen of SC-QuickStart for displaying the QuickStart object manipulation menu:

 

Modify

Opens the QuickStart object creation wizard for the current object.

 

Disable

Disables the QuickStart object. When a object is disabled the linked hotkey will be inactive so that it can be used for other functions. The object can only be executed from the main screen. Right-click the object again and select Enable to re-enable it.

 

Move

Arrange the QuickStart objects with the move commands. This affects the sequence of the QuickStart objects in the main screen as well as the systemtray-menu.

 

Remove

Removes the QuickStart object. Warning: This operation cannot be undone.
