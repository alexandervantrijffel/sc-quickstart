How to execute QuickStart objects


SC-QuickStart offers three ways to QuickStart tasks:

 

Hotkeys

A hotkey requires the least effort for executing QuickStart objects. When the icon of SC-QuickStart is visible in the systemtray (set of icons next to the clock), hotkeys will function anytime. In other words: no matter what application is working on the foreground hotkeys will always be intercepted and handled by SC-QuickStart.

 

Main screen

Execute a QuickStart object from the main screen by clicking on it with the left-mouse button. Use SC-QuickStart's own hotkey CTRL+F1 to show the main screen and click on the desired QuickStart object. Press CTRL+F1 again to hide SC-QuickStart's main screen.

 

Systemtray-menu

When SC-QuickStart is loaded SC-QuickStart's icon will be visible in the systemtray on the bottom right of your screen. Click on it with the right-mouse button to display the systemtray-menu from which the QuickStart objects can be selected.
