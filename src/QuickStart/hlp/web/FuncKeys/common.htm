Common Windows function keys and hotkeys


On this page frequently used keycombinations are listed. These key-combinations should not be used for QuickStart objects because the key-combination will be intercepted by SC-QuickStart before the active application can handle the command.

 

Key-combination Action
CTRL+C Copy selection to clipboard
CTRL+X Cut selection
CTRL+V Paste selection from clipboard
   
CTRL+N New document
CTRL+S Save document
CTRL+P

Print document
CTRL+W Close document
   
CTRL+Z Undo
CTRL+Y Repeat
   
WIN-START+E Explore my computer
WIN-START+F Search for files and folders
WIN-START+R Run program
WIN-START+D Minimize all windows/restore
WIN-START+M Minimize all windows
   
ALT+TAB Switch windows
CTRL+SHIFT+Esc Task manager
